# 第二屆國防應用無人機挑戰賽 — 比賽細則

**全名：** 第二屆國防應用無人機挑戰賽（UAV Defense Challenge）  
**主題：** 無GNSS環境執行多目標偵蒐  
**規章發布：** 中華民國113年9月10日  
**Source:** `114rule.pdf`

---

## 挑戰技術核心

本次比賽模擬無人機受到GNSS干擾無法定位的情境，要求在不使用GPS、不使用FPV（第一人稱視角）的條件下：

- 無GNSS自主導航
- 自主飛行與動態路徑規劃
- 機載目標偵蒐、AI影像辨識與定位

目標：取得目標物之**數量**與**位置座標**。

## 無人機規格要求

- 符合民航局及相關單位規範
- 無GNSS情況下可自主導航，並具備FPV操作與地面控制站返航功能
- 遙控器、無線通訊（含圖傳）、地面控制站使用合法設備與頻段
- **地面控制站顯示需求：**
  - 無人機即時飛行座標與高度（對地高度）；飛行高度限制 200呎～400呎
  - 即時且清楚之目標物辨識結果
  - HDMI輸出至主辦單位螢幕（供裁判評分）
  - 安裝HDMI影音擷取卡（錄製實況操作畫面）
- **禁止使用中國（大陸、港、澳）廠牌之定位、資訊與通訊等產品**
- 多旋翼無人機：斷訊時應具備原地懸停或安全降落能力

**GNSS干擾器（主辦單位提供，必須配合安裝）：**

| 規格項目 | 規格 |
|---------|------|
| 頻段 | L1 1500–1600 MHz & L2 1200–1300 MHz |
| 發射頻率 | GPS、BDS、GLONASS、伽利略 |
| 有效射程 | 1–5 公尺 |
| 重量 | 250 克（含電源） |
| 尺寸 | 115 × 65 × 55 mm（不含天線），天線長 70 mm |
| 固定方式 | 3M子母扣 + 魔鬼氈束帶 |

## 決賽細則

**賽程：** 預計民國114年2月中旬公告。

**飛行區域：** 以嘉義亞洲創新無人機中心為起降點，飛行距離中心半徑5公里內；**部分目標物會放置於飛行區域往外推300公尺範圍內**。

### 決賽目標物

決賽共 **10 個目標**（6個在飛行區域內，4個在飛行區域外300m範圍內），分為**4種類型**：

---

#### 1. 民用車輛（公路局核可車輛）

- 一般乘用車（如 Toyota Altis）
- 靜止狀態
- 車頂放置本賽事 Logo 供辨識
- 從無人機俯視角度：車輛呈矩形輪廓，車頂可見 Logo

---

#### 2. 軍用車輛（悍馬車、吉普車）

- 悍馬車（HMMWV / Humvee）
- 吉普車（Military Jeep）
- 靜止狀態
- 軍綠色迷彩塗裝
- 從俯視角度：車輛比民用車寬，車頂結構較複雜

---

#### 3. 軍事裝備（50加侖油桶）

- 將 **3–6 個** 50加侖油桶堆疊為**一個目標**
- 油桶尺寸：直徑 570 mm × 高度 890 mm
- 顏色不拘（藍色、綠色等）
- 從俯視角度：圓形截面，群集排列

---

#### 4. 兩棲登陸裝甲車（圖形或模型）

- 05式兩棲登陸裝甲車（圖形或實體模型）
- 迷彩塗裝（綠色迷彩或藍色迷彩）
- 特徵：履帶、砲管、車體較長
- 從俯視角度：長形車體，可見砲塔與砲管

---

### 決賽計分方式

**總分上限：150分**（10個目標 × 15分）

**計分公式：**

```
ID_total = Σ (γᵢ + γᵢ/5 × εᵢ)    for i = 1..N

γᵢ = 5   辨識成功
γᵢ = 0   辨識失敗

εᵢ = 10  目標定位誤差 ≤ 50 m
εᵢ = 8   目標定位誤差 ≤ 100 m
εᵢ = 6   目標定位誤差 ≤ 150 m
εᵢ = 4   目標定位誤差 ≤ 200 m
εᵢ = 2   目標定位誤差 ≤ 250 m
εᵢ = 0   其他

N = 10（目標總數）
D = 0.05 × 逾時秒數
R = 1   返航成功
R = 0   返航失敗

各任務得分 = (ID_total - D) × R
```

**同分時：** 以總任務執行時間較低者勝出。

**單目標最高分：** 辨識成功(5) + 5/5 × 10 = **15分**

## 終止任務與比賽

**以下情形由裁判宣布終止該趟任務，視為失敗不予計分：**
- 自行或因故關閉干擾器
- 無人機未以自主飛行方式執行任務
- 超出飛行區域與高度限制（200呎～400呎）

**以下情形由裁判宣布終止比賽：**
- 違反競賽規則
- 干擾競賽進行或違反公平性
- 有安全性疑慮或不依照裁判指示
- 地面控制站與遙控器皆無法與無人機連線
- 無人機不具飛行能力或有飛行安全疑慮

## 對本專案的影響

| 比賽需求 | 本專案現狀 | 差距 |
|---------|-----------|------|
| 無GNSS自主導航 | AnyLoc + VO 視覺定位（完成） | 需整合ArduPilot自主飛行 |
| 民用車輛辨識（初賽，3輛） | YOLOv8n COCO pretrained | COCO有car類別，但需驗證俯視角效果；車頂Logo辨識需額外處理 |
| 民用車輛辨識（決賽） | 同上 | 同上 |
| 軍用車輛辨識（悍馬、吉普） | YOLOv8n無此類別 | 需fine-tune或自訂模型 |
| 油桶辨識 | YOLOv8n無此類別 | 需fine-tune |
| 兩棲裝甲車辨識 | YOLOv8n無此類別 | 需fine-tune |
| 目標定位座標輸出 | AnyLoc定位精度 ~15–20m | 定位誤差需 ≤50m 得滿分；現有精度理論上可達εᵢ=10 |
| 飛行高度回傳 | 已有 agl_m in meta | 需整合至地面站顯示 |
| HDMI輸出顯示 | matplotlib 視窗 | 需確認可HDMI輸出 |

### 關鍵結論

- **初賽（僅民用車輛，3輛）：** YOLOv8n COCO pretrained 的 `car` 類別在俯視角度下可能有效，但需實測驗證。車頂Logo是重要輔助線索。
- **決賽（4種目標類型）：** 軍用車輛、油桶、裝甲車均不在COCO類別中，必須收集航拍訓練資料並fine-tune模型。
- **定位精度：** AnyLoc ~15–20m誤差 → 理論上可達 εᵢ=10（≤50m），接近滿分定位分數。
- **最大優勢：** 無GNSS視覺定位（AnyLoc + VO）正是本次比賽核心挑戰，本專案已有完整實作。
